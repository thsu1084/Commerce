## 프로젝트 기능<br>


### 회원가입과 로그인기능<br>
 * 회원가입<br>
   * 회원가입시 이메일,비밀번호 그리고 역할(소비자 인지 판매자 인지)에 관한 정보가 필요하다<br>
 * 판매자 로그인<br>
   * 판매자 로그인 시 상품을 등록, 삭제, 수정 할 수 있다<br>
### 상품 검색 기능<br>
  * 상품을 검색하면 상품에대한 자세한 정보를 볼 수 있다.<br>
### 상품 관리 기능<br>
  * 상품정보를 저장할 수 있다<br>
  * 상품정보을 삭제할 수 있다<br>
  * 상품의 이름을 수정 할 수 있다<br>
### 장바구니 기능<br>
  * 상품을 장바구니에 담을 수 있다<br>
  * 장바구니에 담긴 상품을 삭제 할 수 있다.<br>
